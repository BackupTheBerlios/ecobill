Hier werden ihre Fragen zum Kundenbereich beantwortet

1. Erstellen von neuen Kunden:

Durch drücken des Button "Erstellt einen neuen Kunden", werden die Textfelder gelöscht und man kann so einen neuen Kunden eingeben.

2. Abspeichern von neuen Kunden:

Geben sie alles notwendige ein und drücken sie den Button "Speichert aktuellen Kunden", schon wird der Kunde in der Tabelle unten abgelegt und steht ihnen zur Verfügung.

3. Löschen von Nachrichten:

Durch drücken des Button "Löscht aktuellen Kunden", wird der durch Sie in der Tabelle ausgewählte Kunde gelöscht.

4. Aktualisieren:

Durch drücken des Button "Aktualisieren" wird die Tabelle und die Textfelder aktualisiert.

5. Auf gehts zum Lieferschein GUI / Auf gehts zum Rechnungs GUI:

Durch Auswahl eines Kunden werden die Buttons "Auf gehts zum Lieferschein GUI" / "Auf gehts zum Rechnungs GUI" freigeschalten und man kann in diesen den Lieferschein bzw. die Rechnung machen.
