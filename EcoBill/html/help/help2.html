Hier werden ihre Fragen zum Artikelbereich beantwortet

1. Erstellen von neuen Artikeln:

Durch drücken des Button "Erstellt einen neuen Artikel", werden alle Textfelder gelöscht und man kann so einen neuen Artikel eingeben.

2. Abspeichern von neuen Artikeln:

Geben sie alles notwendige ein und drücken sie den Button "Speichert aktuellen Artikel", schon wird der Artikel in der Tabelle unten abgelegt und steht ihnen zur Verfügung.

3. Löschen von Artikeln:

Durch drücken des Button "Löscht aktuellen Artikel", wird der durch Sie in der Tabelle ausgewählte Artikel gelöscht.

4. Aktualisieren:

Durch drücken des Button "Aktualisieren" wird die Tabelle und die Textfelder aktualisiert.
